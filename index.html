ScroleyTrigger

Scroll triggered animation for HTML elements

Fade Down Example

By adding the trigger class "sr" followed by the animation type "fade-down" you can trigger a nice fade down animation on a containing div element.

<div class="sr fade-down"></div>

Fade Up Example

Paragraph fades up with a 0.5s delay set by giving the element the class "delay-1"

<p class="sr fade-up delay-1"></p>

What... More delays?!

Out of the box you have up to 10 delay classes each with a 0.5s delay. Just use "delay-1", "delay-2" etc. More delay classes can be easily added via the Sass loop.

<span class="sr delay-2"></span>

Default Variables

This package contains default variables to make your animations consistent. "$transform-distance" and "$transform-ease" can be tweaked to suit your project requirments. More variable tweaks coming soon...

$transform-distance: 64px;
$transform-ease: 0.8s;
